Symfony Shop
Github Project

I developed the Symfony Shop application, an e-commerce platform built on Symfony framework. Leveraging Symfony 7.0 and PHP 8.2.3, I created a user-friendly interface for browsing products. The API endpoint provides real-time product availability information, ensuring users have up-to-date data. Additionally, the admin panel I implemented facilitates easy management of products, users, and cart editing.

One of the key features of my application is the password reset system, enhancing user security and convenience. Support for JavaScript ES14 enables dynamic interactions, enriching the user experience. During the development phase, one of the main challenges I faced was designing the front-end, especially for a backend developer like myself. However, I overcame these challenges, creating a functional and appealing interface. The application is optimized for performance and flexibility, allowing for easy scaling and adaptation to user needs. Furthermore, I utilize Docker containers with Adminer and MySQL, providing not only convenience in database management but also ensuring a consistent development environment.

Symfony OAuth2
Github Project

In my Symfony OAuth2 application, the primary goal was to elevate security standards by implementing OAuth2 authentication. Rather than relying on the built-in authentication system, I opted to leverage OAuth2 for enhanced security. This allowed users to log in to the application seamlessly through Google authentication.

Additionally, I designed a custom authentication system tailored to the specific needs of the application, ensuring robust security measures throughout. The cornerstone of this application is the seamless integration of Google login functionality. To support this feature, I meticulously designed a database schema optimized for OAuth2 authentication, ensuring efficient and secure user authentication processes.

Built on Symfony 7.0, the application embodies modern development practices and utilizes the latest Symfony features to deliver a seamless user experience. Despite encountering challenges during the implementation of OAuth2, I successfully overcame them, ensuring that the authentication process functions flawlessly.
